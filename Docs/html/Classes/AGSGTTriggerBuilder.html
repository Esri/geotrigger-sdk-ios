AGSGTTriggerBuilder Class Reference

Inherits from NSObject
Declared in AGSGTTriggerBuilder.h

Overview

Helps construct an NSDictionary that can be used with AGSGTApiClient to create a trigger.

See the (external) API reference for the trigger/create route for more information.
